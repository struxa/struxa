# Struxa Vision (`struxa-theme`)

**Canonical source:** [github.com/struxa/struxa](https://github.com/struxa/struxa) → `themes/struxa-theme/` in the CMS monorepo.

This folder ships with Struxa CMS. Version bumps and PageSpeed changes are committed to **`struxa/struxa`**, not the legacy [struxa/struxa-theme](https://github.com/struxa/struxa-theme) repository (that repo is not updated for releases).

**Production catalog (struxapoint.com):** rebuild `public/struxa-dist/zips/struxa-theme.zip` from `themes/struxa-theme/` on the server, then **Admin → Struxa catalog → Regenerate catalog**. See `struxa-dist/PUBLISH.md`.
