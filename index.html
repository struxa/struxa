This file is not the marketing homepage

The canonical public marketing site is rendered with Twig when you run the CMS (route GET / → templates/page/home.twig, body sections under templates/partials/marketing/). Header, footer, and shell come from templates/public/root.twig and templates/partials/_header.twig / _footer.twig.

Open the app (for example php -S localhost:8080 -t public public/index.php or Docker) and visit /. Do not duplicate landing markup here — it will drift.

Optional static snapshot (same HTML as live /): run composer render-marketing-preview → writes public/marketing-preview.html.
